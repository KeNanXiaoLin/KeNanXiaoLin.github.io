Front Matter

每篇文章开头都需要以下 front matter（YAML）：

---
title: "文章标题"
date: 2026-07-14T10:00:00+08:00
draft: false  # 改为 false 才会发布
tags: ["标签1", "标签2"]
categories: ["分类"]
---

字段说明：

  • title — 文章标题（必填）
  • date — 发布日期
  • draft — 是否为草稿，发布前保持 false
  • tags — 标签列表
  • categories — 分类
  • description — 描述，用于 SEO 和 RSS
